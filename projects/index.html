Poku

🐷 Como criador de dependências, a ideia inicial era criar um test runner que garantisse de forma fiel se meus projetos funcionavam para Node.js, Deno e Bun ao mesmo tempo, usando a mesma suíte de testes para isso. Então surgiu uma ideia no caminho: E se eu tornasse os testes fáceis?

Como resultado, surgiu um test runner com zero configurações, que vai nas origens do JavaScript e traz de volta a essência da sintaxe nativa para os testes, além de performar até 5 vezes mais que o test runner mais popular de todo ecossistema JavaScript em um benchmark simples.

MySQL2

🐬 Tenho orgulho de ser co-mantenedor do MySQL2, um dos clientes MySQL mais performáticos de todo o ecossistema JavaScript e o mais baixado para Node.js, Bun e Deno, além de ser utilizado por mais de 1 milhão de repositórios públicos no GitHub.

Ele é o principal componente por traz de projetos como Sequelize, Drizzle ORM, TypeORM, Knex, entre outros frameworks (ORM) para conexões MySQL.

Em 24 de março de 2024, migrei todo o conjunto de testes do MySQL2 para o Poku, garantindo a compatibilidade entre Node.js, Bun e Deno.

SVPS: Simplifying VPS

🚀 A ideia original era apenas automatizar algumas tarefas pessoais quando comprava uma VPS ou EC2 "zerada", mas acabou virando um ORM extremamente robusto para servidores Ubuntu.

Além de reparar e instalar as ferramentas mais usadas em servidores de forma granular, o SVPS também permite criar proxies automaticamente, direcionando portas para qualquer domínio ou subdomínio que você quiser.

Atualmente, esse projeto é uma das fontes majoritárias do lucro da minha empresa.

AWS SSL Profiles

📜 Originado de uma confusão após a AWS mudar todos seus certificados de autenticação dos serviços RDS, quebrando sistemas que funcionavam perfeitamente há anos, esse projeto veio para unificar os certificados em um único lugar.

Dessa forma, qualquer mantenedor de um projeto que dependa desses certificados (mysql, mysql2 e pg, em especial), não precisa mais se preocupar com atualizações inesperadas.

Apesar de sua simplicidade, ele é baixado mais de 6 milhões de vezes por mês.

Packages Update

📦 Depois de notar que todos os atualizadores de pacotes NPM possuiam diversas dependências, além de chegarem a megabytes, esse projeto iniciou como um desafio pessoal de criar o mesmo, mas sem nenhuma dpendência.

Atualmente, o projeto evoluiu e é usado especialmente para atualizar monorepositórios dinâmicamente, pesando cerca de 40KB no total final de sua instalação.

Poku Theme

🎨 Um tema fofo em tons "gothic pastel" inspirado na identidade visual do Poku test runner.

Enquanto o Poku test runner facilita os testes, o tema do Poku facilita a leitura do seu código, pois não se trata apenas de combinar cores, mas de estabelecer consistência entre diferentes sintaxes de linguagens.

Pode verificar aí com seu tema favorito se os tipos em TypeScript e PHP têm a mesma cor, porque no Poku Theme vai 😏
