🙋🏻‍♂️ Quem é Weslley Araújo?

Weslley Araújo é um desenvolvedor com mais de 10 anos de experiência, co-mantenedor do MySQL2 — o driver MySQL mais usado no ecossistema JavaScript — e criador do Poku, um test runner de alta performance e o único compatível com todos os runtimes JavaScript da atualidade. Reconhecido como Microsoft MVP, ele impacta milhões de projetos e desenvolvedores globalmente através do Open Source.

🗺️ São Paulo, SP — Brasil.
👷🏻 Empresa: weslley.io.
🔧 Cargo: Principal Developer (& Founder).

💡 Principais Projetos e Iniciativas

  • 🐷 Poku:
    • O único test runner compatível com todos os runtimes para JavaScript no backend da atualidade (incluindo TypeScript).
    • Traz a sintaxe nativa do JavaScript para os testes, democratizando o acesso a testes automatizados para desenvolvedores de todos os níveis.
  • 🔥 lru.min:
    • Implementação inteligente, absurdamente performática e com alta compatibilidade para cacheamento baseado em LRU para JavaScript.
  • 🐬 MySQL2 (Co-mantenedor):
    • O cliente MySQL mais performático e baixado para Node.js, Bun, Deno e Cloudflare Workers, utilizado por mais de 1 milhão de projetos públicos no GitHub.
  • 🐜 Awesome You:
    • Uma iniciativa que quebra a barreira da popularidade, trazendo um ambiente justo ao reconhecer mantenedores open source por diversas métricas de impacto no mundo real.
  • 🐢 NodeBR (Co-organizador e Community Manager):
    • Uma comunidade que promove o desenvolvimento e a colaboração entre desenvolvedores de Node.js e seu ecossistema no Brasil.
