Poku

🐷 As a dependency creator, the initial idea was to create a test runner that would truthfully ensure that my projects worked for Node.js, Deno and Bun at the same time, using the same test suite to do so. Then an idea came to me along the way: What if I made testing easy?

As result, a zero-configuration test runner has arisen, which goes back to the JavaScript origins and brings back its native syntax essence for tests, plus performing up to 5 times faster than the most popular test runner in the entire JavaScript ecosystem in a simple benchmark.

MySQL2

🐬 I'm proud to be a co-maintainer of MySQL2, one of the faster MySQL clients in the entire JavaScript ecosystem and the most downloaded for Node.js, Bun, and Deno, as well as being used by more than 1 million public projects on GitHub.

It's the main component behind projects such as Sequelize, Drizzle ORM, TypeORM, Knex, among other frameworks (ORM) for MySQL connections.

On March 24, 2024, I migrated the entire MySQL2 test suite to Poku, ensuring compatibility between Node.js, Bun and Deno.

SVPS: Simplifying VPS

🚀 The original idea was just to automate some personal tasks when I purchased a “zeroed” VPS or EC2, but it ended up becoming an extremely robust ORM for Ubuntu servers.

As well as repairing and installing the most commonly used tools on servers in a granular approach, SVPS also enables you to create proxies automatically, directing ports to any domain or subdomain you want.

This project is currently one of the main sources of my company's profits.

AWS SSL Profiles

📜 Originated from a confusion after AWS changed all its authentication certificates for RDS services, breaking systems that had been working perfectly for years, this project came about to unify the certificates in one place.

This way, any maintainer of a project that depends on these certificates (mysql, mysql2 and pg in particular) no longer has to worry about unexpected updates.

Despite its simplicity, it's downloaded more than 6 million times a month.

mysqljs/aws-ssl-profiles

Packages Update

📦 After noticing that all the NPM package updaters had several dependencies, as well as megabytes, this project began as a personal challenge to create the same one, but without any dependencies.

Today, the project has evolved and is used especially to update monorepositories dynamically, weighing around 40KB in the final total of its installation.

Poku Theme
